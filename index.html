Load

Provide the JSON file containing J-card data. A saved J-card usually has the .jcard.json filename extension.

Save

This saves the J-card data into JSON text as a file download, whose file name follows either the upper or lower title. No data will be sent to any server.

Due to platform limitations, cover images must be saved and loaded manually.
