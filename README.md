# J-Card Template

[Open the app](https://fabiotavernini.github.io/jcard-template/)

## Building

Check `root` in `/_data/_.yaml`, then:

    $ jekyll serve

### PWA

Also check `scope` and `start_url` in the manifest, and `PATH_PAGE` in the
service worker.
